# Restaurant Page

Project from The Odin Project.

## Goal

Practice building a website using:
- JavaScript DOM manipulation
- ES Modules (import/export)
- Webpack bundling
- Dynamic rendering (no multi-page HTML)

## How it works

The site is fully generated with JavaScript.

A single container (`#content`) is used, and pages (Home, Menu, Contact) are rendered by replacing its content instead of reloading HTML pages.

Navigation is handled through JS.

## Build

Webpack outputs the final files into the `dist/` folder.

Normally `dist/` is ignored in `.gitignore`, but it may be included here for easy preview of the final build.
